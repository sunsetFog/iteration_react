# iteration_react
start

react-cli创建React项目: https://blog.csdn.net/weixin_34191734/article/details/91404021
1：安装react-cli脚手架: npm install -g build-react
2: 使用脚手架搭建项目: build-react init my-react
3：启动项目
    cd 项目名称   进入到项目的路径下
    npm install  安装项目依赖
    npm start    运行项目
4： 打包项目  npm run build
